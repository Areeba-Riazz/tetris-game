# tetris-game

A polished version of my 1st-semester Programming Fundamentals (PF) project. Built using C++ and SFML.

## Overview
This project was originally developed as part of the PF course at University. It is now being refined to showcase fundamental programming concepts like:
- Arrays and Matrices (for the game grid)
- Functions and Modular Programming
- Structs and Custom Data Types
- File I/O (for High Score persistence)
- Event-driven programming with SFML

## Project Structure
- `src/`: Core logic and main entry point.
- `include/`: Header files for game definitions.
- `assets/`: Textures and fonts used in the game.
- `docs/`: Original project requirements and documentation.
